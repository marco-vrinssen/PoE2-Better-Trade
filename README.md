# PoE2 Better Trade

Enhanced trading experience for the Path of Exile 2 trade site.

A Tampermonkey userscript that adds fuzzy search, item copying, and filter duplication to improve trade workflow.

## Features

- **Fuzzy Search** — Type naturally in search fields with automatic fuzzy matching
- **Item Copy** — Copy item stats to clipboard with one click for easy sharing
- **Filter Duplication** — Duplicate stat filter groups to quickly build complex queries

## Installation

### Tampermonkey (Recommended)

1. Install [Tampermonkey](https://www.tampermonkey.net/) for your browser
2. [Click here to install the script](https://github.com/marco-vrinssen/PoE2-Better-Trade/raw/main/PoE2-Better-Trade.js)
3. Click **Install** in Tampermonkey
4. Visit [PoE 2 Trade](https://www.pathofexile.com/trade2/) to use it

### Manual Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/)
2. Click the Tampermonkey icon → Create a new script
3. Copy the contents of `PoE2-Better-Trade.js` into the editor
4. Save
