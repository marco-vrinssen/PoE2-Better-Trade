# PoE2 Better Trade

Enhanced trading experience for Path of Exile 2's trade site with fuzzy search, item copy, and filter duplication features.

## Features
- **Fuzzy Search**: Quickly find items using fuzzy matching in search fields.
- **Item Copy**: Copy item stats to clipboard with a single click.
- **Filter Group Duplication**: Duplicate stat filter groups for faster query building.

## Installation (Tampermonkey)
1. Install the [Tampermonkey extension](https://www.tampermonkey.net/) for your browser.
2. Click "Create a new script" in Tampermonkey.
3. Copy the contents of `PoE2-Better-Trade.js` into the editor.
4. Save the script.
5. Visit [Path of Exile 2 Trade](https://www.pathofexile.com/trade2/) and enjoy enhanced trading!
